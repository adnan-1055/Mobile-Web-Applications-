Task 1: Cute Icons App

An interactive Android app built in Java using XML layouts, animations, and MediaPlayer.
Users can tap on different animal icons to trigger:

1) Sound effects
2) Shake animation
3) Dynamic text & colour updates
4) Reset button that returns UI to default

Features

1) 10 animal icons
2) Shake animation using shake.xml
3) MediaPlayer for audio playback
4) Reusable setIconListener() method
5) Reset button with dynamic styling
6) Clean UI built with XML

Tech Stack

1) Java
2) Android Studio
3) XML Layouts
4) MediaPlayer
5) AnimationUtils
6) R resources

What I Learned

1) Creating reusable listeners
2) Integrating animations in Android
3) Managing MediaPlayer lifecycle
4) Updating UI dynamically
5) Handling user interaction with XML + Java
